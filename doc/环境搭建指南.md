# 环境搭建指南（Windows）

## 一、前置软件安装

### 1. Python（>= 3.13）
- 下载地址：https://www.python.org/downloads/
- 安装时勾选 **"Add Python to PATH"**
- 验证：
  ```bash
  python --version
  ```

### 2. uv（Python 包管理器）
- 安装命令（PowerShell）：
  ```powershell
  powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
  ```
- 验证：
  ```bash
  uv --version
  ```

### 3. Node.js（>= 18）
- 下载地址：https://nodejs.org/
- 选择 LTS 版本安装
- 验证：
  ```bash
  node --version
  npm --version
  ```

## 二、启动后端服务

```bash
# 进入 server 目录
cd server

# 创建虚拟环境并安装依赖
uv sync

# 启动服务（默认端口 5000）
uv run python app.py
```

启动成功后终端会显示 `Running on http://127.0.0.1:5000`。

默认管理员账号：`admin` / `123456`

## 三、启动前端服务

新开一个终端窗口：

```bash
# 进入 client 目录
cd client

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

启动后访问终端中显示的地址（通常为 `http://localhost:5173`）。

## 四、常见问题

| 问题 | 解决方法 |
|------|----------|
| `uv` 命令未找到 | 重新打开终端，或手动将 uv 安装路径加入系统 PATH |
| `npm install` 速度慢 | 设置镜像：`npm config set registry https://registry.npmmirror.com` |
| 端口 5000 被占用 | 修改 `server/app.py` 中的 `port` 参数 |
| 端口 5173 被占用 | Vite 会自动切换到下一个可用端口 |
